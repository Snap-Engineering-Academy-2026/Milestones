# Milestone 2 - Final Project Starter - Partner Work

Now that we have worked through a smaller bug-fix/issue and worked through some fun git challenges, we ready are to tackle a bigger feature in groups of 2! 

## Before You Code
On Google Classroom submit your wireframes of your feature ("Milestone 2 Wireframe"). This can be done on you computer (turn in screenshots) or with pen and paper(turn in photos). This must be submitted prior to coding!

## Feature Guidelines  
1. The feature should involve at least one new screen (to practice with navigation)
2. The feature should involve reading and writting data from supabase(to practice with databases)
3. Make an effort to be style/ make it look nice

## Contributing Best Practices 
1. Create your own branch, the name of the branch should make sense to what you are working on.
2. Pay close attention to your git ensuring that you are maintaining your branch up to date before merging. 
3. Pair programing when needed.
4. Delegate and work in different files whenever possible.
5. Make space for others and their ideas.
